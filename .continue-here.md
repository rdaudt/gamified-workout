<decisions_made>

- Product direction is a free multi-coach platform, not a single-trainer branded app.
- Guests can do the full session flow, including download/share, but nothing persists afterward.
- One account can carry multiple roles: trainee, coach, admin.
- A trainee can have zero or one current coach in MVP.
- Coach selection is strictly current-coach-or-app, never manual per card/session.
- Historical workouts/cards keep the branding and CTA context they had when created.
- If no coach is assigned, cards use app branding and the UI may prompt directory discovery instead of a coach booking CTA.
- Approved coaches control directory visibility separately from approval status.
- Repo noise and generated media should stay out of version control going forward.
</decisions_made>
