Product Design — Section 2

System components — what it's made of

How the pieces fit together
User layer
📱
Web App
Runs in any browser. Camera, microphone, and screen — no install. Works on phone or laptop.
React · responsive design
🎤
Voice Commands
Listens for "Start" and "Done" during a session. Hands-free — you never touch the screen mid-workout.
Web Speech API (built into browser)
🪪
Accounts
Guest mode for first try. Free account to save history. No payment, no friction.
Email / Google login
processed locally on device
AI layer (on device)
🦴
Pose Detection
Tracks 33 body landmarks in real time using the camera. Draws the skeleton overlay. Runs entirely in the browser — nothing uploaded.
MediaPipe Pose (Google)
🔢
Rep Counter
Measures joint angles (elbow, hip, knee) to detect full vs. partial reps. Each exercise has its own angle thresholds.
Custom rules per exercise type
📐
Form Analyser
Flags alignment issues: back sagging, knees caving, shallow depth. Records which reps had form breaks. Generates post-session feedback.
Angle + symmetry thresholds
rep data sent to server — no video
Effort engine (server)
📊
History Analyser
Compares today vs. all previous sessions for this exercise. Detects plateaus, streak length, and rate of improvement over time.
⚡
Effort Score
A score (1–100) reflecting the real difficulty of today's improvement. A plateau-breaking +3% scores higher than an easy +40%.
✍️
Quote Generator
Generates the "past self" quote calibrated to the context: breakthrough, first win, plateau, consistency. Unique every time.
Claude API
stored in database
Data layer
💾
Workout History
One record per session: exercise, date, rep count, form score, effort score. Small and private — never includes video.
🃏
Card Store
Generated cards stored as images. Shareable via URL or download. Your trainer branding baked in to every card.
🔔
CTA Engine
Monitors form flags and plateau patterns. Triggers personalised coaching CTAs when thresholds are met. Points to your booking link.
